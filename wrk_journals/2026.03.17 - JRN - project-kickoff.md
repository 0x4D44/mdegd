# Journal: Project Kickoff - Eric's Birthday Quiz

**Date:** 2026-03-17
**Project:** Birthday quiz for Eric (Arthur's dad), playable at a URL via GitHub Pages

## Context
- Eric's birthday: March 21st
- He likes: US state history, bridge (card game), card cheating/sleeve jokes, Scottish hydro-electric power (NSHEB)
- All questions must relate to events on March 21st

## Key Design Decisions

### Quiz Mechanics
- Pool of ~100 questions, 12 randomly selected per play
- Name selection: Eric (hard mode) vs Ruth (easy/comedy mode)
- Eric mode: very similar answers, nitpicky distinctions
- Ruth mode: one obvious answer, rest are nonsense (or occasionally all the same)

### Visual Design
- Pool of visual styles, one chosen at random per session
- Birthday-themed but varied

### Tech Stack
- Single-page static HTML/CSS/JS
- No dependencies — works offline, easy to host
- GitHub Pages for publishing

### Development Approach
- Red-Green TDD
- Agent team with devil's advocate
